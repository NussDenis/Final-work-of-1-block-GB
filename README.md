## __Описание решения итоговой проверочной работы__

### _**Задача**: Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа_
1. Объявление массива строк произвольной длины элементов.
2. Вывод массива на печать в консоль.
3. Запуск цикла перебора каждого элемента в массиве.
4. Создание переменной счетчика.
5. Запуск цикла перебора каждого символа в элементе массива с увеличеснием счетчика при каждой иттерации.
6. Создание условия при выполнении которого элементы массива, с содержанием символов менее либо равным 3-м, выводятся на печать в консоль.
